## 介绍
基于`JDK17`、`WebFlux`、`R2DBC`开发的后台管理系统。接口采用RESTful风格，并支持Docker多阶段构建。前端代码：https://github.com/NotEdibleSalt/ContainsLightAdmin

## 功能模块
- 用户管理
- 角色管理
- 权限管理
- 菜单管理
- 字典管理
- 文章管理
- 登录记录
